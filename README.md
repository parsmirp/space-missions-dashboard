# Space Missions Dashboard

An interactive dashboard to visualize and analyze historical space mission data from 1957 onwards, built with R Shiny.

## Live Dashboard
https://parsmirp.shinyapps.io/space_mission_project/

## How to Run Locally
1. Clone this repo
2. Install dependencies in R:
```r
install.packages(c('shiny','shinydashboard','DT','ggplot2','plotly','dplyr','lubridate','readr'))
```
3. Run the app:
```r
shiny::runApp()
```
